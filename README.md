# Student Performance Analysis 📊

## 📌 Project Overview
This project analyzes student performance using basic data science techniques.

## 🔧 Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## 📊 What I Did
- Cleaned the dataset
- Calculated average marks
- Created Total and Percentage columns
- Visualized data using bar graphs and histograms

## 📈 Insights
- Highest scoring student: Priya
- Most students scored between 70–90%
- Science has slightly higher average marks

## ✅ Conclusion
This project helped understand student performance patterns using simple data analysis and visualization techniques.
